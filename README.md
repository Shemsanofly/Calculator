Calculator Programs (C++)
Overview

This folder contains my very first set of programs written in C++, focusing on building different versions of a calculator. These projects were created as part of my journey in learning programming fundamentals such as input/output, conditional statements, loops, and basic problem-solving techniques.

The calculator is one of the most common beginner projects in programming, and it helped me understand how logic is applied to perform mathematical operations in code. Through these versions, I gradually improved my skills and explored different ways to implement the same idea with increasing complexity.

Purpose of This Project
The main goal of these calculator programs is to:
Build a strong foundation in C++ programming
Practice logical thinking and problem-solving
Understand how simple programs can evolve into more complex systems

How to Run
Option 1: Using Command Line (g++)
Make sure you have a C++ compiler installed (like g++).
Open terminal or command prompt.
Navigate to the folder containing your .cpp file.

Compile the program:

g++ filename.cpp -o calculator

Run the program:

./calculator
Option 2: Using Dev-C++

If you are using Dev-C++, follow these steps:

Open Dev-C++
Click on File → Open and select your .cpp file
Click on Execute → Compile & Run
Shortcut: F11
The program will compile and run automatically
Output will appear in the console window
Notes
Make sure your file has .cpp extension
Always save your file before compiling
If there are errors, check the compiler messages and fix them

Conclusion

These calculator versions represent my learning progress in C++. Starting from a simple program and gradually adding features helped me understand how software is built step by step.
